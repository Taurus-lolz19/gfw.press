## Installation

### From PyPI

\`\`\`bash
pip install gfw.press
\`\`\`

### From Source

\`\`\`bash
git clone https://github.com/${GITHUB_USER}/gfw.press.git
cd gfw.press
pip install -e .
\`\`\`

## Quick Start

\`\`\`python
import gfw.press

# Initialize the client
client = gfw.press.Client()

# Use the main functionality
result = client.process_data("your_data_here")
print(result)
\`\`\`
